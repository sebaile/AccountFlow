## 📖 About

**AccountFlow** is a web application designed to simplify the management of streaming accounts and subscriptions.

It provides a clean and intuitive interface to organize customer information, monitor subscription dates, calculate profits, and manage account data stored in Excel.

Perfect for freelancers, resellers, and small businesses looking for an organized and practical administration tool.

---

## ✨ Features

* 📋 Manage streaming accounts
* 👥 Customer management
* 📅 Subscription expiration tracking
* 💰 Automatic profit calculation
* 📊 Dashboard with statistics
* 📄 Excel integration (OpenPyXL)
* 🔍 Search and filter accounts
* 📱 Responsive interface
* ⚡ Fast and lightweight

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AccountFlow.git
```

Enter the project folder:

```bash
cd AccountFlow
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
